# Storage Decisions

## Cloud Storage
- Purpose: Store unstructured log files
- Use case: Backup, raw logs

## Cloud SQL
- Purpose: Store relational data (Users)
- Use case: Applications needing SQL queries

## Spanner
- Purpose: Global relational database
- Use case: High availability, multi-region apps

## Firestore
- Purpose: Store app sessions
- Use case: Real-time apps

## Bigtable
- Purpose: Store metrics and time-series data
- Use case: Monitoring large datasets
